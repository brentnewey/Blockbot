# Blockbot

A Unity 2D puzzle game featuring grid-based movement and clever puzzle mechanics.

## Play Online

Play Blockbot in your browser at: https://brentnewey.github.io/blockbot-web/

## About

Blockbot is a puzzle game where you navigate through levels using strategic movement and interact with various game objects including:
- Pushable blocks
- Boxing gloves for punching
- Crossbows and projectiles
- Bomb robots (bombots)
- And more puzzle mechanics!

## Controls

- Arrow keys or WASD to move
- Z to undo
- R to restart level
- Space to use items (e.g., punch with boxing glove)
